All-in-one AI content creation suite for WordPress — generate posts, Web Stories, automation workflows, and smart publishing tools in one click.

== Description ==

Alpha Suite is the ultimate suite for content creators, marketers, and WordPress website owners who want to create content quickly, intelligently, and in an optimized way.

Funcionalidades:

* Posts GPT — gere artigos completos automaticamente, otimizados para SEO e Google Discover
* Alpha Stories — crie Web Stories perfeitas com um clique, direto do editor
* Painel unificado — gerencie licença, módulos, preferências e API Keys
* Integração nativa com OpenAI
* Planejamento automático de conteúdo
* IA embarcada para criação e otimização

== Installation ==

1. Envie o plugin para a pasta `/wp-content/plugins/alpha-suite/`
2. Ative pelo menu “Plugins”
3. Acesse “Alpha Suite” para ajustar suas configurações e ativar sua licença

== Frequently Asked Questions ==

= Necesito OpenAI API Key? =  
Sim, o plugin recomenda uma chave OpenAI para gerar conteúdo e Web Stories.

= Posso usar meus próprios prompts? =  
Sim, existe uma área dedicada para personalização.
